# SkyBank Frontend

A premium digital banking interface built with React, Vite, Tailwind CSS, and Ant Design.

## Tech Stack
- **Framework:** Vite + React (JSX)
- **Styling:** Tailwind CSS + Vanilla CSS
- **Components:** Ant Design (antd)
- **Icons:** Ant Design Icons + Lucide React
- **Animations:** Framer Motion
- **API Client:** Axios (Interceptors for JWT)

## Features
- **Secure Authentication:** Login and Registration with JWT persistence.
- **Dynamic Dashboard:** Overview of net worth, monthly growth, and recent transactions.
- **Account Management:** View all accounts (Savings, Current, FD) and open new ones.
- **Transactions:** High-fidelity forms for Money Transfers, Deposits, and Withdrawals.
- **Activity Statement:** Detailed searchable and filterable transaction history.
- **Premium UI:** Glassmorphism, smooth transitions, and responsive layout.

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Dev Server:**
   ```bash
   npm run dev
   ```

3. **Backend Configuration:**
   Ensure your API Gateway is running on `http://localhost:8089`.
